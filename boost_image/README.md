## Installation

Steps:

1. `./bootstrap.sh --prefix=path/to/installation/prefix`
> https://www.boost.org/doc/libs/1_81_0/more/getting_started/unix-variants.html#easy-build-and-install

2. `./b2 install`

3. `./b2 headers`
